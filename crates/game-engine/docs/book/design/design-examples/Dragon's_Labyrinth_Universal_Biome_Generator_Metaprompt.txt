# Dragon's Labyrinth Universal Biome Generator Metaprompt

## Core Template Structure

Generate a low-poly 3D hexagonal terrain tile optimized for Dragon's Labyrinth RPG in Godot 4.x.

### UNIVERSAL TECHNICAL SPECIFICATIONS
- **Format**: .glb with embedded materials and vertex colors
- **Geometry**: Regular hexagon, flat-top orientation, 2 units diameter  
- **Polygon Budget**: 200-500 triangles (varies by complexity)
- **Pivot**: Centered at base for seamless grid snapping
- **Material Setup**: Single material slot named "{BiomeName}Material"
- **LOD Support**: Include 2-3 LOD levels (full, medium 50%, simple 25%)
- **Edge Treatment**: Standardized connection points for transitions
- **Godot Integration**: Vertex colors, unlit materials, convex collision hull

## DYNAMIC GENERATION PARAMETERS

### Biome Intensity Levels
- **Subtle**: Minimal decorative elements, clean base geometry
- **Standard**: Moderate detail, 3-5 decorative objects  
- **Rich**: High detail, 6-10 elements, complex surface variation
- **Epic**: Maximum detail, unique landmarks, story elements

### Transition Support
- **Edge Blending**: Smoothed borders for adjacent biome mixing
- **Height Transitions**: Gradual elevation changes (Â±0.5 units)
- **Color Gradients**: Vertex color blending zones
- **Element Bridging**: Shared decorative elements across borders

### Seasonal Variations
For any biome, append:
- `SEASON: Spring` - Fresh growth, bright colors, budding elements
- `SEASON: Summer` - Full bloom, vibrant, maximum vegetation  
- `SEASON: Autumn` - Fading colors, fallen elements, harvest themes
- `SEASON: Winter` - Snow coverage, ice, dormant vegetation, cold blues

### Lighting Conditions  
- `LIGHTING: Daylight` - Bright, clear visibility, sharp shadows
- `LIGHTING: Twilight` - Purple/orange tints, long shadows, mystical atmosphere
- `LIGHTING: Underground` - No natural light, torch-friendly, dark ambiance
- `LIGHTING: Magical` - Unnatural colors, glowing elements, ethereal effects

This metaprompt system can replace the existing asset libraries by:

1. **Batch Generation**: Generate complete biome sets (20-50 tiles per biome)
2. **Automatic Integration**: Generate corresponding GDScript libraries
3. **Procedural Variation**: Create 3-5 variants per biome type
4. **Transition Sets**: Generate edge pieces for smooth biome blending
