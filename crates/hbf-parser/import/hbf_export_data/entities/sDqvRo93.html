Description

Trap!
There's a spring-loaded spears trap here, hidden inside shafts in the floor. When undetected using a DC17 Wisdom (Perception, passive or active) check, it will trigger if removing a specific object from its original place. Succeed a DC20 Dexterity saving throw or sustain 2d4 damage from the violent thrust of the jumping spears.
