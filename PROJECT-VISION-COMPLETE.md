# PROJECT VISION COMPLETE - Dragon's Labyrinth

## THE GAME WE'RE BUILDING
A 180-level horror experience using RPG mechanics to deliver personal tragedy through mechanically varied, emotionally progressive gameplay.

## THREE-PART STRUCTURE

### PART 1: Journey TO (Levels 1-60)
**Mechanical Arc**: Learn systems through organic discovery
**Emotional Arc**: Hero → Doubt → Horror → Broken
**Key Innovation**: 2.5D caves → 3D dungeons → DOOM labyrinths

### PART 2: Journey FROM (Levels 61-120)  
**Mechanical Arc**: Systems corrupted and subverted
**Emotional Arc**: Guilt → Desperation → Leadership → Hope?
**Key Innovation**: World changed by dragon's death

### PART 3: Seal Void (Levels 121-180)
**Mechanical Arc**: Transcend systems, create reality
**Emotional Arc**: Madness → Acceptance → Transformation → Seal
**Key Innovation**: You become what you destroyed

## CORE GAMEPLAY LOOP

### The Rhythm (Every ~6 Levels)
```
HEX OVERWORLD (Tactical/Strategic)
    ↓
3D LABYRINTH (Action/Horror)
    ↓
BOSS ARENA (Unique Mechanics)
    ↓
REPEAT WITH VARIATION
```

### Why This Works
- **Variety**: Never 3 levels without mechanical change
- **Engagement**: Switching between tactical and action
- **Horror**: Architecture as enemy in 3D sections
- **Memory**: Set pieces players won't forget

## MECHANICAL PROGRESSION

### Organic Discovery (Never Forced)
1. **Height Advantage**: Discovered by climbing hill
2. **Sound Navigation**: Emerges from darkness
3. **Weapon Switching**: Boss patterns require it
4. **Sanity Management**: Madness becomes tool
5. **Reality Creation**: Void requires imagination

### System Evolution
- **Movement**: Walk → Run → Climb → Float → Phase
- **Combat**: Melee → Tactical → Sanity → Reality
- **Navigation**: Map → Sound → Memory → Creation
- **Companions**: Allies → Relationships → Trauma → Shared Madness

## THE 3D LABYRINTH IDENTITY

### Using 1.7GB CC0 Assets
**Medieval Models (85MB)**: Base architecture
**Horror Models (112MB)**: Corruption overlays
**Environmental (386MB)**: World building

### Progressive Corruption
1. **Clean**: Medieval dungeons with torches
2. **Unsettling**: Shadows wrong, doors lead nowhere
3. **Corrupted**: Horror blends with medieval
4. **Nightmare**: Non-Euclidean, breathing walls
5. **Void**: Geometry rebels, you create path

### DOOM-Style Evolution
- **Early**: Classic key/door progression
- **Mid**: Multi-level vertical complexes
- **Late**: First-person horror sections
- **Dragon**: Stalked through shifting maze
- **Void**: No rules, only intention

## EMOTIONAL JOURNEY

### Through Mechanics
- **Peace**: "I'm a hero!" (standard RPG)
- **Unease**: "Something's wrong" (systems glitch)
- **Dread**: "This isn't right" (horror appears)
- **Terror**: "I can't trust anything" (reality shifts)
- **Despair**: "What have I done?" (dragon truth)
- **Madness**: "Reality is optional" (void emerges)
- **Void**: "I am the seal" (become guardian)

### Philosophy Paths (Mechanical Differences)
**Strength/Harmony** (First Half):
- Strength: Command armies, tactical sacrifice
- Harmony: Build bonds, shared resources

**Light/Dark** (Second Half):
- Light: Cleansing mechanics, ward creation
- Dark: Corruption harvesting, void powers
